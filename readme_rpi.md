# Installation #

Just unpack archive contents to your home folder.

Note : use 50 Hz screen modes for full synchronization.

You can set it in /boot/config.txt

720p mode:

```
hdmi_group = 1
hdmi_mode  = 19
```

1080p mode:

```
hdmi_group = 1
hdmi_mode  = 31
```

See http://elinux.org/RPiconfig for more info.

# Controls #

## Keys in emul ##
  * ESC			- menu.
  * Arrows, CTRL	- joystick.
  * F12 - reset
  * F7 - pause
  * F2 - quick save
  * F3 - quick load
  * F5 - start/stop tape
